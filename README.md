# Portfolio Website

## Overview

This is my personal portfolio website built using HTML and CSS. It showcases my skills, projects, and interests in Software Development, Artificial Intelligence, and Cloud Computing.

## Features

* Responsive single-page design
* Hero, About, Skills, Projects, and Contact sections
* Clean and modern user interface
* External links to GitHub and LinkedIn profiles
* Custom cloud-themed design

## Technologies Used

* HTML5
* CSS3
* Git & GitHub
* AWS S3 (Static Website Hosting)
* AWS CloudFront (HTTPS & CDN)
